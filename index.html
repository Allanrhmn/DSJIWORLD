Configuration

Customize the extension in VS Code Settings (Ctrl+,):

Setting Type Default Description
spidjb.refreshInterval number 5 Refresh interval in minutes (1-1440)
spidjb.showInStatusBar boolean true Show price in status bar
spidjb.alertThreshold number 0 Alert when change exceeds % (0 = disabled)
spidjb.statusBarPosition string right Position: "left" or "right"

Example Configuration

{ "spidjb.refreshInterval": 3, "spidjb.showInStatusBar": true, "spidjb.alertThreshold": 1.5, "spidjb.statusBarPosition": "right" }

Troubleshooting

Price Not Updating?

  • Check VS Code Output channel: "SPIDJB Stock Viewer"
  • Verify your internet connection
  • Try manually refreshing: SPIDJB: Refresh Price
  • Restart VS Code

Wrong Price Data?

  • The extension parses Nordnet's page structure
  • If the page layout changes, parsing may fail
  • Report issues on GitHub

Alerts Not Working?

  • Ensure spidjb.alertThreshold is greater than 0
  • Set it to your desired percentage threshold (e.g., 1.5 for ±1.5%)
